Save your notes into the repo (notes.json) so they survive a cache clear and follow you across devices. Paste a GitHub token with permission to write to this one repo — it's stored only on this device.

  1. Open github.com → new fine-grained token
  2. Under Repository access → Only select repositories → choose News-Notes
  3. Under Permissions → Repository → Contents, set Read and write
  4. Click Generate token, copy it, and paste below
