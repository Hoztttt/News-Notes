# News Notes 📰

A dead-simple app for remembering the news. Each day, jot down a quick note
about **3 stories** you read. Writing them down (and being able to look back)
helps them stick.

## How to use

Just open **`index.html`** in your browser — that's it. No install, no server,
no account.

- **Today** tab: fill in up to 3 stories (a headline + a note on why it matters
  / what you want to remember) and hit *Save*.
- **History** tab: browse everything you've saved, grouped by day, with a
  search box.

## Where's my data?

Everything is stored **locally in your browser** (`localStorage`) on the device
you use — nothing is uploaded anywhere. It stays private to you.

> Tip: If you want it on your phone's home screen, open the file in your mobile
> browser and use "Add to Home Screen." To sync across devices you'd need to
> host the file somewhere (e.g. GitHub Pages) — the notes themselves still live
> per-device.
